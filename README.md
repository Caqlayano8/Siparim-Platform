## 🏗️ Proje Yapısı

```
siparim/
├── backend/      # NestJS API
├── frontend/     # Next.js Web (Müşteri + Restoran + Admin)
├── mobile/       # Flutter (Müşteri + Kurye Uygulaması)
└── docs/         # Teknik belgeler
```

## 🚀 Özellikler

- 📱 Müşteri mobil uygulaması (Flutter)
- 🛵 Kurye uygulaması (Flutter)
- 🏪 Restoran yönetim paneli
- 🧑‍💻 Admin paneli
- ⚡ Gerçek zamanlı sipariş takibi (Socket.IO)
- 💳 Online ödeme (İyzico / PayTR)
- 🗺️ Google Maps entegrasyonu
- 🔔 Push bildirimler (FCM)
